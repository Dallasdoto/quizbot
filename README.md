# Quiz Bot для Telegram

Бот для создания и проведения викторин. Правильный ответ всегда корректно сохраняется.

## Команды
- /start — начало работы
- /newquiz — создать новую викторину
- /myquizzes — список викторин
- /startquiz — запустить викторину
- /deletequiz — удалить викторину

## Запуск на Railway
1. Загрузи файлы на GitHub
2. Подключи репозиторий в Railway
3. Добавь переменную окружения: BOT_TOKEN=твой_токен
4. Deploy!
